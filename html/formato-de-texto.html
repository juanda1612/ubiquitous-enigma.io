Elementos de formato de texto

Los elementos de formato se diseñaron para mostrar tipos especiales de texto:

  • <b>- Texto en negrita
  • <strong>- Texto importante
  • <i>- Texto en cursiva
  • <em>- Texto enfatizado
  • <mark>- Texto marcado
  • <small>- Texto más pequeño
  • <del>- Texto eliminado
  • <ins>- Texto insertado
  • <sub>- Texto subíndic
  • <sup>- Texto en superíndice

Elementos HTML <b> y <strong>

El elemento HTML <b> define texto en negrita, sin importancia adicional.

HTML
<b>Este texto está en negrita</b>

El elemento HTML <strong> define el texto con gran importancia. El contenido del interior normalmente se muestra en negrita.

HTML
<strong>Este texto es importante!</strong>

Elementos HTML <i> y <em>

El elemento HTML <i> define una parte del texto en una voz o estado de ánimo alternativo. El contenido interior normalmente se muestra en cursiva.

HTML
<i>Este texto esta en cursiva</i>

El elemento HTML <em> define el texto enfatizado. El contenido interior normalmente se muestra en cursiva.

HTML
<em>Este texto esta enfatizado</em>

Elementos HTML <small>

El elemento HTML <small> define texto más pequeño:

HTML
<small>Este es un texto más pequeño</small>

Elementos HTML <mark>

El elemento HTML <mark> define el texto que debe marcarse o resaltarse:

HTML
<p>No olvides comprar <mark> leche </mark> hoy</p>

Elementos HTML <del>

El elemento HTML <del> define el texto que se ha eliminado de un documento.

HTML
<p>Mi color favorito es <del>azul</del> rojo</p>

Elementos HTML <ins>

El elemento HTML <ins> define un texto que se ha insertado en un documento. Los navegadores normalmente subrayarán el texto insertado:

HTML
<p>Mi color favorito es <del>azul</del> <ins>rojo</ins> </p>

Elementos HTML <sub>

El elemento HTML <sub> define el texto del subíndice. El texto con subíndice se puede utilizar para fórmulas químicas, como H2O:

HTML
<p>este es un texto <sub>subscrito</sub> </p>

Elementos HTML <sup>

El elemento HTML <sup> define texto en superíndice. Se puede utilizar texto en superíndice para notas a pie de página, como WWW [1] :

HTML
<p>Este es texto en <sup>superíndice</sup></p>

Elementos de citas HTML

En este apartado veremos los elementos <blockquote>, <q>, <abbr>, <abbr>, <cite>y <bdo>

HTML <blockquote> para citas

El elemento HTML <blockquote> define una sección que se cita de otra fuente. Los navegadores suelen aplicar sangría a <blockquote>los elementos.

HTML
<p>Aquí hay una cita del sitio web de WWF:</p>
<blockquote cite="http://www.worldwildlife.org/who/index.html">
Durante 60 años, WWF ha trabajado para ayudar a las personas y a la naturaleza a prosperar. Como organización conservacionista líder en el mundo, WWF trabaja en casi 100 países. En todos los niveles, colaboramos con personas de todo el mundo para desarrollar y ofrecer soluciones innovadoras que protejan a las comunidades, la vida silvestre y los lugares en los que viven.
</blockquote>

HTML <q> para citas breves

El elemento HTML <q> define una cita breve. Los navegadores normalmente insertan comillas alrededor de la cita.

HTML
<p>El objetivo de WWF es: <q> Construir un futuro donde las personas vivan en armonía con la naturaleza.</q></p>

HTML <abbr> para abreviaturas

El elemento HTML <abbr> define una abreviatura o un acrónimo.

HTML
<p>La <abbr title="Organización Mundial de la Salud"<OMS>/abbr> fue fundada en 1948.</p>

HTML <address> para información de contacto

El elemento HTML <address> define la información de contacto del autor/propietario de un documento o artículo.

HTML
<address>
Escrito por John Doe.<br>
Visitanos en:<br>
Ejemplo.com<br>
Box 564, Disneyland<br>
USA
</address>

HTML <cite> para el título del trabajo

El elemento HTML <cite> define el título de una obra creativa.

HTML
<p><cite>El Grito</cite>de Edvard Munch. Pintado en 1893. </p>

HTML <bdo> para anulación bidireccional

El elemento HTML <bdo> se utiliza para anular la dirección del texto actual:

HTML
<bdo dir="rtl">Este texto se escribirá de derecha a izquierda.</bdo>
